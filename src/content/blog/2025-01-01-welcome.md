## About This Site

This website is built with modern web technologies to provide a fast, accessible, and enjoyable reading experience:

- **[Astro](https://astro.build)** - A modern static site generator that delivers lightning-fast performance
- **[TypeScript](https://www.typescriptlang.org/)** - For type-safe, maintainable code
- **[Tailwind CSS](https://tailwindcss.com/)** - For beautiful, responsive styling
- **[MDX](https://mdxjs.com/)** - Allowing rich, interactive content in blog posts

5. **SEO Optimized** - Complete with meta tags, Open Graph, and Twitter Cards.

## How to Add Blog Posts

If you're curious about how this site works (or if you're a future version of me who forgot!), here's how to add a new blog post:

1. Create a new `.md` or `.mdx` file in the `src/content/blog/` directory
2. Add frontmatter with the required fields:
   ```yaml
   ---
   title: "Your Post Title"
   description: "A brief description"
   date: 2025-01-01
   tags: ["tag1", "tag2"]
   ---
   ```
3. Write your content using Markdown
4. Commit and push - GitHub Actions will automatically build and deploy!
